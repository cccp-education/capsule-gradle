== Détail technique

=== CapsuleConfigMerger (318 lignes)
- `loadFromGradleProperties(projectDir)` — lit `capsule.*` depuis `gradle.properties`
- `loadFromEnvironment()` — lit `CAPSULE_*` env vars
- `merge(projectDir, yamlConfig, cliParams, yamlLoaded)` — merge avec priorité CLI > YAML > Props > ENV
- Paramètre `yamlLoaded: Boolean` distingue "pas de YAML" de "YAML avec defaults"
- Méthodes `*NoYaml` (CLI > Props > ENV) quand `yamlLoaded = false`
- Pattern aligné sur plantuml-gradle ConfigMerger
