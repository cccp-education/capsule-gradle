=== 1. Bug-fix — PlaywrightCaptureImpl.close() avant check fichier vidéo

*Cause racine* : `PlaywrightCaptureImpl` écrit la vidéo dans `RecordVideoDir`
mais le fichier `.webm` n'est finalisé qu'au `context.close()`.
Le code vérifiait l'existence du fichier *avant* le `close()` (l.460-468),
ce qui causait `No video generated by Playwright capture` systématiquement.

*Fix* : `CapsuleVideoTask.kt` l.450-475 — `close()` déplacé dans le bloc
`finally` *avant* le check `listFiles { *.webm }`.
