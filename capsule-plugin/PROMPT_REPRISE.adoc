== Code review — points à auditer prioritairement

=== HAUTE PRIORITÉ (à corriger avant merge)

1. **fail-late TTS** — `CapsuleBuildTask.execute()` :
   Avant : fail-fast (throw dès la première erreur TTS)
   Après : on collecte toutes les erreurs, throw à la fin
   → Vérifier que c'est intentionnel. Si oui, le documenter.
   → Si non, remettre le throw dans le catch (comportement historique).

2. **ManimEngine — pas de isAvailable() upfront** :
   `captureSlideParallel()` appelle `ManimEngine.render()` sans vérifier
   que manim est installé. On découvre l'échec slide par slide.
   → Ajouter ManimEngine.isAvailable() check en pré-vol, avant le dispatch.

=== MOYENNE PRIORITÉ

3. **`ExecutorService.submit<R>` non typé** :
   `Future<*>` au lieu de `Future<File?>` ou `Future<Unit>`.
   → Changer en `executor.submit<File?> { ... }` pour la sécurité de type.

4. **`isPlaceholder()` supprimé sans remplacement** :
   La méthode détectait les fichiers TTS factices (< 500 octets, # TTS PLACEHOLDER).
   Si encore utile, la restaurer ou justifier sa suppression.

=== BASSE PRIORITÉ

5. **`AUDIO_INJECT_SCRIPT` hardcodé** :
   35 lignes de JS multilignes dans le companion object.
   → Externaliser dans `src/main/resources/audio-inject-script.js`.

6. **`createSingleSlideHtml()` lit/écrit le deck N fois** :
   24 slides = 24 lectures + 24 écritures du fichier HTML complet.
   → Lire une fois en RAM, générer les N variantes, écrire en batch.

7. **`ManimEngine.render()` — `mediaDir.walk()` lent** :
   DFS sur tout le dossier media → préférer le chemin standard Manim :
   `mediaDir/videos/{quality}/{sceneName}.mp4`

8. **`ScreenshotCaptureImpl` accepte `List<Double>` mais n'utilise que `firstOrNull()`** :
   → Changer la signature en `duration: Double` pour clarifier l'API.

=== TESTS À AJOUTER

9. **Test parallélisme** : vérifier que `captureSlideParallel()` produit les bons
   segments WebM dans le bon ordre, avec concat correct.

10. **Test Manim** : scénario Cucumber "slide [manim:SceneName]" → vidéo produite
    (mock manim si pas installé, ou skip avec @DisabledIfNoManim)

1. git remote add lion92 https://github.com/lion92/capsule-gradle.git
